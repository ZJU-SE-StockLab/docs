## 项目结构

按照课程要求，我们将开发一个 **股票交易系统** 的软件工程项目，包括客户端（可以是 Web 端、也可以通过 Electron 封装的桌面端）与服务端。

我们采取 Schema-First 的设计模式，使用基于 OpenAPI 规范定义的 API 接口约束前后端开发，并将通过代码实现与框架约束实践这一点。

├── client: 客户端代码
├── server: 服务端代码
